## Description:

本APP主要面向于服务高校新生、留学生、校外参访人员，
可以为他们提供亲密友好的校园导航以及校内建筑、
教室使用说明等一系列相关介绍，同时还可以动态更新数据，
让用户对APP有充分的依赖感。当然，我们的APP将具有很高的可扩展性，
只要是在一定区域范围内（景点、商场等）都可以通过技术移植实现相关功能。
